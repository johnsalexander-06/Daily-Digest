7-Zip patches actively dangerous remote-code-execution flaw

The widely used 7-Zip archive tool released an update fixing a heap-based buffer-overflow vulnerability that attackers could exploit to run arbitrary code. Because 7-Zip is ubiquitous on Windows systems and frequently unmanaged, the flaw presents a broad attack surface. Patching relies on users updating manually, so exposure will linger well past disclosure.

So What: Unmanaged archive utilities are a classic soft entry point onto DoD and contractor endpoints. IT and cyber staffs should push the 7-Zip update and inventory where it is installed outside standard patch cycles.
